# Dotfiles Agent Entrypoint

This repo keeps the live shared scaffold in `agents/CLAUDE.md`.

When operating inside `dotfiles`:
- Read `agents/CLAUDE.md` first.
- Treat `agents/AGENTS.md` as the Codex-compatible alias.
- Use repo-root `CLAUDE.md` / `AGENTS.md` only as entrypoints that point to `agents/`.
